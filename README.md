## Paper Abstract

Second-hand e-commerce marketplaces in emerging markets face a distinctive security challenge: high fraud exposure combined with limited security infrastructure. Existing IDS solutions are anchored at the network layer and do not model the application-level behavioral patterns most informative for detecting marketplace-specific threats such as credential stuffing, automated catalog scraping, and mass fake account creation.

This paper presents a multi-modal, two-layer intrusion detection architecture designed and validated for **SoukDZ**, an Algerian second-hand fashion marketplace. The system decouples the detection engine from the marketplace application through a structured Apache log stream. Layer 1 analyzes network-level flows trained on CICIDS2017 (957,452 samples). Layer 2 analyzes application behavioral logs generated by executing five attack scenarios from a Kali Linux VM against a Flask mock. Random Forest achieves **F1 = 1.0000 and AUC = 1.0000** on both layers. A five-scenario operational validation demonstrates detection within 10 seconds and **zero false positives** against simultaneously running legitimate traffic. SHAP-based attribution provides per-alert explanations in a Streamlit dashboard.
